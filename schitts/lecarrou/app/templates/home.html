Schitt's Creek Cafe Tropical sale platform



Info

This apps/API have been developped using Django framework/Python and Docker.

This API is able to efficiently:
  • store all customer's informations: Schitt's Creek Cafe Tropical's customers can create an account to send comment
  • store all customer's orders: see database diagram at https://dbdiagram.io/d/5fd0815c9a6c525a03ba5f6f
  • store all the menu items available: using www.thecocktaildb.com API to create 'fake' datas
  • store the customer feedback once bill is paid
Goals achieved:
  • demonstrate recovery from a complete crash of all instances: using asynchronous tasks with Celery and Celery-beats to make scheduled backup every day at midnight
  • simulate the insertion of 10.000 customer making between 2 to 14 orders per year on a span of 2 years: using Faker for generating dates, www.thecocktaildb.com API for drinks and api.spoonacular.com for courses.
